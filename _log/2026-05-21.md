## Summary
* More progress on *QMIX: Monotonic Value Function Factorisation for Deep Multi-Agent Reinforcement Learning* by Rashid et. al.

## Notes
### Literature.
1. Alright, so
    * Each agent is represented by its own network,
    * There's a feedforward network that combines all Q values into a joint Q value - this is called the mixing network.
    * The weights of the mixing network are restricted to be strictly non-negative - this is to enforce monotonicity, which guarantees the individual global max property.
    * A **hypernetwork** is employed here - the weights of the mixing network themselves are generated by a neural network. Hmmm... this can allow for non-linear relationships between the agents that are *state-dependent*. 
2. I'm a little fuzzy on the idea of the individual global max property... it's a consistency measure, seemingly, that ensures that if an individual agent takes an action that increases its local Q-value, the global joint-value will never decrease.
3. Ahhh, all this is saying is that the optimal joint action chosen by a group of agents together is exactly the same as the collection of optimal actions chosen by each agent individually.
4. I want to spend a bit more time with this paper. It's giving me GNN vibes: I need to digest the functional approximation mechanisms.
