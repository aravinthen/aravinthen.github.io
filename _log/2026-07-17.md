## Summary
1. Built a simple variational autoencoder. 

## World models
1. I used a few medium tutorials for building the VAE, mostly because the introductory paper that I read was very much based around the theory. However, these things aren't so bad - granted, I have no clue what I'm going to see when I inject some images into it. 
2. Now, the core difference here is - *how exactly do I control a VAE in the light of heterogeneity?* What does that even look like? All of the environments I'm working with are quite stochastic - the wall generator uses percolation. 
3. I think I'm going to use a common dataset, or better yet use a very simple problem using my existing environment before I launch into using VAEs for world modelling. 
4. Goal for this project: *reproduce World Models by Schmidhuber and Ha*.
